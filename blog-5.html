Operating Systems

System Software Last updated: April 2024

What is an Operating System?

An OS is system software that manages hardware resources and provides services to application programs.

Process Management

A process is a program in execution. Key concepts include:

  • Process States: New, Running, Waiting, Ready, Terminated
  • Context Switching: Save/restore process state
  • Process Scheduling: FCFS, SJF, Round Robin, Priority
  • Deadlock: When processes block each other

Memory Management

Virtual Memory

Extends physical memory using disk space for efficient multi-tasking.

Paging and Segmentation

Paging divides memory into fixed blocks, segmentation into variable-sized blocks.

Page Replacement Algorithms

FIFO, LRU (Least Recently Used), Optimal page replacement

File Systems

Organizes data storage on disks:

  • File organization and naming
  • Directory structure
  • Access control (permissions)
  • File allocation methods (contiguous, linked, indexed)

Synchronization and IPC

Critical Section: Code that accesses shared resources

Semaphores: Synchronization primitives

Mutex: Mutual exclusion locks

Message Passing: Inter-process communication

Tip: Understanding OS concepts helps write efficient and safe concurrent programs.
