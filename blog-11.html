Blog #11

Artificial Intelligence

AI Last updated: April 2024

Introduction to AI

AI is about creating machines that can perform tasks requiring human intelligence like reasoning, learning, and problem-solving.

AI vs ML

AI: Broader field - making intelligent machines

ML: Subset of AI - learning from data

Search Algorithms

Uninformed Search

  • BFS: Explores level by level
  • DFS: Explores as deep as possible first
  • Uniform Cost Search: Explores by cost

Informed Search

  • Greedy Best-First: Uses heuristic to pick promising node
  • A* Search: Combines actual cost and heuristic

Heuristics

Problem-specific estimates that guide search algorithms to goal faster. Examples: Manhattan distance, Euclidean distance.

Game Theory

Minimax: Tree search algorithm for two-player games

Alpha-Beta Pruning: Optimization technique for minimax

Monte Carlo Tree Search: Probabilistic search for complex games

Knowledge Representation

  • Logic: Propositional and first-order logic
  • Frames: Structured knowledge representation
  • Semantic Networks: Graph-based knowledge
  • Ontologies: Formal knowledge structure

Expert Systems

Systems that use domain expert knowledge encoded as rules to make decisions.

Components: Knowledge base, inference engine, explanation module

NLP (Natural Language Processing)

Processing human language:

  • Tokenization and parsing
  • Sentiment analysis
  • Named entity recognition
  • Machine translation

Computer Vision

Understanding and interpreting visual information from images and videos.

Tip: AI is rapidly evolving - stay updated with latest research and applications.
