React & Frontend Frameworks

Frontend Last updated: April 2024

What is React?

JavaScript library for building user interfaces with reusable components and reactive data flow.

Core Concepts

Components

Reusable UI building blocks. Can be class-based or functional.

function Welcome() {
    return <h1>Hello, World!</h1>;
}

JSX

Syntax extension allowing HTML-like code in JavaScript.

Props

Parameters passed to components for configuration.

State

Mutable data within component that triggers re-render when changed.

React Hooks

  • useState: Add state to functional components
  • useEffect: Handle side effects
  • useContext: Access context values
  • useReducer: Complex state logic
  • useCallback: Memoize functions
  • useMemo: Memoize values

Component Lifecycle

Mounting: constructor → render → componentDidMount

Updating: render → componentDidUpdate

Unmounting: componentWillUnmount

Routing

React Router: SPA navigation with react-router-dom

Define routes and navigate between pages without page reload.

State Management

  • Context API: Built-in solution for prop drilling
  • Redux: Centralized state management
  • MobX: Reactive state management
  • Zustand: Lightweight state management

Other Popular Frameworks

  • Vue.js: Progressive framework, easy to learn
  • Angular: Full-featured framework by Google
  • Svelte: Compiler-based framework
  • Next.js: React framework with SSR and static generation

Performance Optimization

  • Code splitting and lazy loading
  • Image optimization
  • Memoization (React.memo)
  • Virtual scrolling for large lists
Tip: Build small components for better reusability and testing.
