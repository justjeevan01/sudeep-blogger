Database Management Systems

Database Last updated: April 2024

What is DBMS?

A Database Management System is software that manages databases, allowing users to create, read, update, and delete data efficiently.

Types of Databases

Relational (SQL)

Data organized in tables with relationships. Examples: MySQL, PostgreSQL, Oracle

Non-Relational (NoSQL)

Flexible schema, key-value stores. Examples: MongoDB, Cassandra, Redis

SQL Fundamentals

SELECT: Retrieve data from tables

INSERT: Add new records

UPDATE: Modify existing records

DELETE: Remove records

JOIN: Combine data from multiple tables

Database Normalization

Process of organizing data to reduce redundancy:

  • 1NF: Atomic values, no repeating groups
  • 2NF: No partial dependencies
  • 3NF: No transitive dependencies
  • BCNF: Every determinant is a candidate key

Transactions and ACID Properties

  • Atomicity: All or nothing
  • Consistency: Data integrity maintained
  • Isolation: Concurrent transactions don't interfere
  • Durability: Committed data persists

Indexing and Query Optimization

Indexes speed up data retrieval. Types include primary key index, composite index, and full-text index.

Remember: Always backup your database regularly and design schemas carefully.
