Cybersecurity Essentials

Security Last updated: April 2024

Introduction to Cybersecurity

Cybersecurity is the practice of protecting systems, networks, and information from cyber attacks.

CIA Triad

  • Confidentiality: Information accessible only to authorized people
  • Integrity: Data accuracy and consistency maintained
  • Availability: Systems accessible when needed

Common Threats

  • Malware: Viruses, worms, trojans, ransomware
  • Phishing: Social engineering to steal credentials
  • SQL Injection: Malicious SQL commands in input
  • DDoS Attacks: Overwhelming system with traffic
  • Man-in-the-Middle: Intercepting communication
  • Zero-day Exploits: Unknown vulnerabilities

Cryptography

Symmetric Encryption

Same key for encryption and decryption. Examples: AES, DES

Asymmetric Encryption

Public and private key pairs. Examples: RSA, ECC

Hashing

One-way function to verify data integrity. Examples: MD5, SHA-256

Authentication and Authorization

Authentication: Verify user identity (passwords, MFA, biometrics)

Authorization: What authenticated user can access

SSL/TLS

Secure communication protocol for HTTPS. Encrypts data in transit.

Network Security

  • Firewalls: Filter traffic based on rules
  • VPN: Secure remote access
  • IDS/IPS: Intrusion detection and prevention
  • WAF: Web application firewall

Best Practices

  • Use strong, unique passwords
  • Enable multi-factor authentication (MFA)
  • Keep systems and software updated
  • Regular backups of important data
  • Security awareness training
  • Principle of least privilege (PoLP)
Remember: Security is not a destination but a continuous process. Stay vigilant!
