Computer Networks

Networking Last updated: April 2024

OSI Model (7 Layers)

Framework for understanding network communication:

  • Layer 1 - Physical: Cables, signals, voltages
  • Layer 2 - Data Link: MAC addresses, Ethernet
  • Layer 3 - Network: IP routing
  • Layer 4 - Transport: TCP, UDP
  • Layer 5 - Session: Connection management
  • Layer 6 - Presentation: Encryption, compression
  • Layer 7 - Application: HTTP, FTP, SMTP

TCP/IP Protocol Suite

TCP (Transmission Control Protocol): Reliable, ordered delivery

UDP (User Datagram Protocol): Fast, connectionless

IP Addressing

IPv4: 32-bit addresses (192.168.1.1)

IPv6: 128-bit addresses for future scalability

Subnetting: Dividing networks using subnet masks

HTTP & HTTPS

HTTP: Stateless protocol for web communication

HTTPS: HTTP with SSL/TLS encryption for security

Methods: GET, POST, PUT, DELETE, PATCH

DNS (Domain Name System)

Translates domain names to IP addresses. Uses hierarchical naming system.

Network Security

  • Firewalls and ACLs
  • VPN (Virtual Private Network)
  • Cryptography and SSL/TLS
  • DDoS protection
Remember: Understanding network layers helps diagnose connectivity issues effectively.
