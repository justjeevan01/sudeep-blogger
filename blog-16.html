Node.js & Backend Development

Backend Last updated: April 2024

What is Node.js?

JavaScript runtime for server-side development, enabling full-stack JavaScript development.

Event-Driven Architecture

Node.js uses non-blocking I/O and event-driven model for efficient handling of multiple requests.

Express.js Framework

Minimal web framework for building REST APIs and web applications.

const express = require('express');
const app = express();

app.get('/', (req, res) => {
    res.send('Hello, World!');
});

Routing

Define HTTP endpoints with different methods (GET, POST, PUT, DELETE).

Middleware

Functions that execute during request-response cycle:

  • Logging
  • Authentication
  • Error handling
  • CORS management

Authentication

  • Session-based: Store user info on server
  • JWT (JSON Web Tokens): Stateless authentication
  • OAuth 2.0: Third-party authentication

Database Integration

SQL Databases

Sequelize: ORM for MySQL, PostgreSQL

NoSQL Databases

Mongoose: MongoDB object mapper

Async Operations

  • Callbacks: Traditional approach (callback hell)
  • Promises: Better structure
  • Async/Await: Cleaner syntax for async code

REST API Design

  • Use HTTP methods properly (GET, POST, PUT, DELETE)
  • Meaningful URL paths
  • Proper status codes (200, 201, 400, 404, 500)
  • JSON request/response format
  • Version your API

Popular Libraries

  • dotenv: Environment variables
  • bcryptjs: Password hashing
  • joi: Data validation
  • multer: File uploads
Tip: Always validate and sanitize user input for security.
