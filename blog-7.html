Python Programming

Programming Language Last updated: April 2024

Why Python?

Python is popular for its simple syntax, readability, and powerful libraries. Used in data science, web development, AI, and automation.

Python Basics

  • Variables: Dynamic typing, no declaration needed
  • Data Types: int, float, str, list, dict, tuple, set
  • Control Flow: if/else, for, while loops
  • Functions: def keyword, parameters, return values

Object Oriented Programming

Python supports classes and objects:

class Dog:
    def __init__(self, name):
        self.name = name
    
    def bark(self):
        return f"{self.name} says Woof!"

Important Libraries

  • NumPy: Numerical computing
  • Pandas: Data manipulation
  • Matplotlib: Data visualization
  • Django/Flask: Web frameworks
  • Requests: HTTP library
  • Beautiful Soup: Web scraping

File Handling

Reading and writing files with open() context manager:

with open('file.txt', 'r') as f:
    content = f.read()
    print(content)

Exception Handling

Handle errors gracefully with try-except blocks:

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero!")

List Comprehensions

Elegant way to create lists:

squares = [x**2 for x in range(10)]
Tip: Use virtual environments (venv) to manage project dependencies.
