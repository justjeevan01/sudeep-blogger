Java Programming

Programming Language Last updated: April 2024

Java Fundamentals

Java is a versatile, object-oriented language with "write once, run anywhere" philosophy through JVM.

Basic Syntax

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Data Types and Variables

  • Primitive: byte, short, int, long, float, double, boolean, char
  • Reference: String, Arrays, Objects
  • Naming: camelCase for variables, PascalCase for classes

Object Oriented Programming

Java is purely OOP. Key concepts:

  • Classes and Objects
  • Inheritance (extends keyword)
  • Polymorphism (method overriding)
  • Encapsulation (getters/setters)
  • Abstract classes and interfaces

Collections Framework

Built-in data structures:

  • List: ArrayList, LinkedList (ordered, allow duplicates)
  • Set: HashSet, TreeSet (unique elements)
  • Map: HashMap, TreeMap (key-value pairs)
  • Queue: PriorityQueue, LinkedList

Exception Handling

try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("Error: " + e.getMessage());
} finally {
    System.out.println("Cleanup");
}

Multithreading

Create concurrent programs with Thread class or Runnable interface.

Popular Frameworks

  • Spring Boot: Web applications
  • Hibernate: ORM (Object-Relational Mapping)
  • Apache Kafka: Event streaming
Tip: Use IDE like IntelliJ IDEA or Eclipse for better development experience.
